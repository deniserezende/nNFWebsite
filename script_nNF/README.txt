- To run the script first "navigate to the location of the script"
cd [path to the script FOLDER]

- To get the path to the folder containing the script file:
Drag the folder to the terminal or command prompt or PowerShell

- Then
1. If in Mac:
	python3 nNF_script.py -p [path to the zip file]

2. If in Windows:
	python nNF_script.py -p [path to the zip file]


- To get the path to the zip file:
Drag the file to the terminal or command prompt or PowerShell


- If the path contains spaces it should look something like this:
1. If in Mac:
	python3 nNF_script.py -p /Users/Denise/Downloads/xml_nfe_2021-11-04-16-08-19.907684\ \(1\).zip
	python3 nNF_script.py -p "/Users/Downloads/xml_nfe_2021-11-04-16-08-19.907684 (1)"

2. If in Windows PowerShell:
	python nNF_script.py -p "C:\Users\denise\Downloads\xml_nfe_2021-11-05-21-23-04.017995.zip (1)"

3. If in Windows Command Pompt
	python nNF_script.py -p "C:\Users\denise\Downloads\xml_nfe_2021-11-05-21-23-04.017995.zip (1)"
